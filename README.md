# R EDA and Web Scraping Project

## Project Objectives
This project demonstrates:
1. Exploratory Data Analysis (EDA) using R built-in datasets
2. Web scraping from a static website using R
3. Data visualization and structured data extraction

## Tools and Technologies
- R Programming Language
- ggplot2
- rvest
- dplyr

## Part 1: Exploratory Data Analysis
- Dataset Used: iris
- Summary statistics and structure analysis
- Visualizations include histogram, boxplot, and scatter plot

## Part 2: Web Scraping
- Website: https://books.toscrape.com/
- Extracted Data:
  - Book titles
  - Prices
  - Ratings
- Data presented in a structured data frame

## How to Run the Code
1. Install required libraries:
   install.packages(c("ggplot2", "rvest", "dplyr"))
2. Run `eda_builtin_dataset.R` for EDA
3. Run `web_scraping_static_site.R` for web scraping
